Installation
============

getML is a software suite for automated feature engineering on
relational data and time series. It enables you to complete your data
science projects in a fraction of their usual time and with better results. 
getML comes with an easy-to-use
Python API that allows you to get started quickly and to start working
on your own data after only a short learning curve.

You can download and install getML on multiple platforms. Please refer
to one of the following sections:


.. toctree::
   :maxdepth: 1

   Mac <mac>
   Windows/Docker <windows>
   Linux <linux>

instructions in
